UltraRAG 已集成多个经典与前沿的 RAG 基线方法，并提供统一的运行与评估接口，便于用户快速复现与对比。当前已支持的基线包括：

- Vanilla LLM
- Vanilla RAG
- CRAG
- DeepNote
- IRCoT
- IterRetGen
- R1-Searcher
- RankCoT
- Search-o1
- Search-r1

更多基线持续更新中。相关实验结果与复现设置将持续在 Leaderboard 页面中维护与更新。

## 快速复现方式

用户可通过以下两步快速复现任意基线方法：

### 1. 构建 Pipeline 配置

```shell
ultrarag build examples/<baseline_name>.yaml
```

### 2. 运行 Pipeline 进行推理与评估

```shell
ultrarag run examples/<baseline_name>.yaml
```

请将 `<baseline_name>` 替换为对应基线方法的 YAML 文件名（如 vanilla_rag.yaml、ircot.yaml 等）。

📌 提示：如需查看所有可用基线的 YAML 文件，可参考项目中的 examples/ 目录。
