title: "UltraRAG 2.0"
description: "UltraRAG 2.0 是首个基于 Model Context Protocol (MCP) 架构设计的轻量级 RAG 框架，面向科研与应用场景提供统一建模。系统将 Retriever、Generator 等核心组件标准化封装为可复用的 MCP Server，并以函数级 tool 接口实现灵活扩展；流程控制由自定义 MCP Client 驱动，开发者只需编写 YAML 即可配置串行、循环与条件分支等复杂逻辑，百行内代码即可实现复杂范式的推理流程。框架附带从入门到精通的保姆级教程，并预集成 17 个主流 benchmark 及多种最新 baseline，配合统一知识库支持可复现、可对比的系统化评测，显著降低开发与调试成本。"
