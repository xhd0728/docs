## UltraRAG 2.0

UltraRAG 2.0 是首个基于 [Model Context Protocol (MCP)](https://modelcontextprotocol.io/overview) 架构设计的轻量级 RAG 框架，专为科研与应用场景提供统一建模能力。系统将 **Retriever**、**Generation** 等核心组件标准化封装为可复用的 MCP Server，并通过函数级 tool 接口实现灵活扩展。流程控制由自定义 MCP Client 驱动，开发者只需编写 YAML，即可配置串行、循环、条件分支等复杂逻辑，百行内代码即可实现复杂范式的推理流程。  

框架附带从入门到精通的保姆级教程，预集成 17 个主流 benchmark 及多种最新 baseline，配合统一知识库支持可复现、可对比的系统化评测，**显著降低开发与调试成本**。

## Why UltraRAG 2.0?

当前 RAG 系统正从简单的模块串联，迈向依赖模型主动推理、动态检索与条件判断的复杂流程。这类系统通常涉及多轮交互与信息更新，传统框架在灵活性和可扩展性方面已难以胜任。同时，现有开源实现缺乏统一范式，不同 baseline 模块耦合严重、结构混乱，极大增加了理解、复现与对比研究的难度。

为应对这些挑战，UltraRAG 2.0 致力于提供：

- 🚀 **低代码构建复杂 Pipeline**  
  原生支持串行、循环、条件分支等推理控制结构。开发者仅需编写 YAML 文件，配合少量自定义代码，即可实现复杂迭代式 RAG 流程（如 Search-O1 等）。

- ⚡ **快速复现与功能扩展**  
  采用 MCP 架构，所有模块均封装为可复用的 Server，支持用户按需自定义 Server 或复用现有模块。每个 Server 的功能通过函数级别的 tool 定义，新增功能只需添加一个函数即可接入流程，显著降低 baseline 复现成本与科研开发门槛。

- 📊 **统一评测与对比**  
  内置标准化的评测流程与指标管理，支持 17 个主流科研 benchmark 的开箱即用与扩展，持续集成最新基线，提供 leaderboard 结果，助力系统性研究与优化。
